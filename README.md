# AURA - Premium Parfüm E-Commerce Web Sitesi

AURA, lüks parfüm ürünlerini sunan modern bir e-commerce web sitesidir. React ve Tailwind CSS kullanılarak tasarlanmıştır.

## 🌟 Özellikler

- **Responsive Tasarım**: Tüm cihazlarda mükemmel görünüm
- **Modern UI**: Tailwind CSS ile güzel arayüz
- **Sayfa Yönetimi**: Ana Sayfa, Ürünler, Hakkımızda, İletişim, Sepet sayfaları
- **Ürün Filtreleme**: Kategoriye göre ürün filtreleme
- **Alışveriş Sepeti**: Ürün ekleme/çıkarma, miktar yönetimi
- **İletişim Formu**: Müşteri iletişim sistemi
- **Profesyonel Tasarım**: Lüks ve modern görünüm

## 🛠️ Teknolojiler

- **React 18**: JavaScript kullanıcı arayüz kütüphanesi
- **Vite**: Hızlı geliştirme sunucusu
- **Tailwind CSS**: Utility-first CSS framework
- **PostCSS**: CSS işlemcisi
- **Autoprefixer**: Otomatik CSS prefix ekleme

## 📁 Proje Yapısı

```
AURA/
├── src/
│   ├── components/
│   │   ├── Header.jsx       # Navigasyon başlığı
│   │   ├── Banner.jsx       # Giriş banner'ı
│   │   └── Footer.jsx       # Alt bilgi
│   ├── pages/
│   │   ├── Home.jsx         # Ana sayfa
│   │   ├── Products.jsx     # Ürünler sayfası
│   │   ├── About.jsx        # Hakkımızda sayfası
│   │   ├── Contact.jsx      # İletişim sayfası
│   │   └── Cart.jsx         # Alışveriş sepeti
│   ├── styles/
│   ├── App.jsx              # Ana uygulama bileşeni
│   ├── App.css              # Uygulama stilleri
│   ├── index.css            # Global stiller
│   └── main.jsx             # Giriş noktası
├── index.html               # HTML şablonu
├── package.json             # Proje bağımlılıkları
├── vite.config.js           # Vite konfigürasyonu
├── tailwind.config.js       # Tailwind konfigürasyonu
└── postcss.config.js        # PostCSS konfigürasyonu
```

## 🚀 Başlangıç

### Kurulum

1. Bağımlılıkları yükleyin:
```bash
npm install
```

### Geliştirme Sunucusu

Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

Tarayıcıda `http://localhost:5173` açın.

### Üretim Derlemesi

Üretim için derleyin:
```bash
npm run build
```

### Önizleme

Derlenmiş siteyi önizleyin:
```bash
npm run preview
```

## 📄 Sayfalar

- **Ana Sayfa**: Koleksiyonlar, neden AURA seçmelisiniz ve yapı bilgileri
- **Ürünlerimiz**: Ürünler listesi, kategoriye göre filtreleme
- **Hakkımızda**: Şirket hikayesi, misyon, vizyon ve istatistikler
- **İletişim**: İletişim bilgileri ve mesaj formu
- **Sepetim**: Alışveriş sepeti yönetimi
